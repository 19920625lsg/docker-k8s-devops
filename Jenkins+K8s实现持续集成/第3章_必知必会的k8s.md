# 第3章 必知必会的k8s

## k8s是什么?

 > Kubernetes 是 Google 开源的一个容器编排引擎,它支持自动化部署、大规模可伸缩、应用容器化管理

## k8s能做什么？

+ 容器的自动化复制和部署,随时扩展或收缩容器规模,并提供负载均衡
+ 方便地容器升级
+ 提供容器弹性,如果失效就替换它

## k8s对于测试能做什么

+ 测试服务器的集中化、自动化管理。将各种平台的服务器加入集群,按需部署或销毁
+ 持续集成时方便地自动部署
