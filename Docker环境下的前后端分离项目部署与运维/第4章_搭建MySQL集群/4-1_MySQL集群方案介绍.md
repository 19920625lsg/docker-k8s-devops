# 4.1 MySQL集群方案介绍

## 常见的mysql集群方案

+ **Replication**：速度快弱一致性低价值适合保存日志新闻和帖子
+ **PXC**：Percona XtraDB Cluster,速度慢，强一致性，高价值适合保存订单账户财务信息。**推荐用此方案**

## PXC方案的数据强一致性

+ **PXC**：**同步复制**,事务在所有集群节点要么同时提交，要么不提交
  ![同步复制](同步复制.jpg)
+ **Replication**：**异步复制**，无法保证数据的一致性
  ![异步复制](异步复制.jpg)
